Installing manual judge
-----------------------

For Arch Linux:

    ./scripts/arch/setup-all-quick.sh
    sudo ./setup.py install --nojail
    # you can ignore the "Please do the following:" message

For Ubuntu:

    cd scripts/ubuntu
    ./setup-all.sh
    cd ../..
    sudo ./setup.py install --nojail
    # you can ignore the "Please do the following:" message
